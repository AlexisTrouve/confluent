## Projet

**Confluent** est un projet de création d'une langue construite (conlang) pour la Civilisation de la Confluence, une civilisation fictive du projet de jeu de rôle **civjdr**.

## Objectif

Créer une langue évolutive cohérente avec plusieurs stades historiques :
- **Proto-Confluence** : Racines primitives (50-80 racines monosyllabiques)
- **Ancien Confluent** : Langue sacrée (150-200 mots)
- **Confluent Classique** : Langue du jeu actuel (300-500 mots)

## Principes linguistiques

### Phonétique (Mix Basque + Finnois)
- Consonnes nettes : k, t, s, z, ts, x (son "ch")
- Voyelles riches : a, e, i, o, u (+ possiblement ä, ö)
- Harmonie vocalique
- Éviter : sons trop elfiques (L/R liquides), sons anglo/latins génériques, consonnes gutturales

### Structure : Liaisons sacrées
Les racines (1 glyphe = 1-2 syllabes) se combinent via des voyelles de liaison porteuses de sens :
- **-i-** : Agent/Porteur (celui qui fait)
- **-a-** : Union/Avec (ensemble)
- **-u-** : Appartenance (de, venant de)
- **-o-** : Devenir/Vers (transformation)
- **-e-** : Source/Origine (né de)

### Exemple
**Siliaska** = sil (regard) + -i- (porteur) + aska (libre) = "Les porteurs du regard libre"
